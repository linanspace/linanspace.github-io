简介：

Markdown 是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档。
Markdown 语言在 2004 由约翰·格鲁伯（英语：John Gruber）创建。
Markdown 编写的文档可以导出 HTML 、Word、图像、PDF、Epub 等多种格式的文档。
Markdown 编写的文档后缀为 .md, .markdown。

Markdown 应用

Markdown 能被使用来撰写电子书，如：Gitbook。
当前许多网站都广泛使用 Markdown 来撰写帮助文档或是用于论坛上发表消息。
例如：GitHub、简书、reddit、Diaspora、Stack Exchange、OpenStreetMap 、SourceForge等。

Markdown标题

在文字前面加#号，就可以把这段文字设置为标题。一个#号是一级标题，两个#号是二级辩题，以此类推，最大支持六级标题

一级标题： # 一级标题
二级标题： # 二级标题
…

六级标题： # 六级标题

注意：标准语法一般是在#号后面加一个空格在写文字！

字体样式

Markdown支持四种文本样式：

  • 加粗： 将要加粗的文字左右两边加两个*号
  • 斜体： 将要倾斜的文字左右两边加一个*号
  • 斜体加粗： 将要倾斜和加粗的的文字左右两边加三个*号
  • 删除线： 将要加删除线的文字左右两边分别加两个 ~ 号

引用

在需要设为引用的文字前面加上>即可，引用也可以嵌套：

  • 一级引用 使用 >加引用文字
  • 二级引用 使用 >>加引用文字
  • 三级引用 使用 >>>加引用文字

······以此类推

图片

在Markdown中插入图片语法如下：

![Alt text](图片链接 “optionaltitle”)

Alt text: 图片的Alt标签，用来描述图片的关键词，可以不写。最初的作用是用来当图片不能

正常显示时出出现的文字说明，后被用SEO，可以方便搜索引擎根据Alt text里面的关键词搜索到图片。

图片链接：图片的本地地址或者是在线网址。

“optional title”:鼠标悬停于图片上出现的标题文字，可不写。

超链接

Markdown支持超链接，语法格式如下：

[超链接名](超链接地址 "超链接title" )

有序列表

需要将文字列表化时，可以在文字前面加上数字和点，并且需要在点后面留有一个空格即可。

效果如下：

  1. 今天又是快乐的一天
  2. 明天又是欢乐的一天
  3. 祝大家每天都能心情愉悦

无序列表

需要将文字列表化时，可以在文字前面加-、+、*这些符号中的任何一种，并且需要在点后面留有一个空格即可。

效果如下：

  • 今天又是快乐的一天
  1. 明天又是欢乐的一天
  2. 祝大家每天都能心情愉悦
