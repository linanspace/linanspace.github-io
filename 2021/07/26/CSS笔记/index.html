1.1什么是CSS

Cascading Style Sheet 层叠级联样式表

CSS：表现（美化网页）包括字体，颜色，边距，高度，宽度，背景图片，网页定位，网页浮动…..

1.2发展史

CSS1.0

CSS2.0 DⅣ（块）+CSS,HTML与CSS结构分离的思想，网页变得简单，SEO

CSS2.1浮动，定位

CSS3.0圆角，阴影，动画…浏览器兼容性~

1.3快速入门

简单入门

  1. 在HTML网页中的head标签中加入style标签在里面写CSS代码

  2. 行内样式

  3. 建议使用外部样式表

优先级

就近原则，谁距离被修饰的近谁的优先级就高

css的优势

  1. 内容和表现分离
  2. 网页结构表现统一，可以实现复用
  3. 样式十分的丰富
  4. 建议使用独立于htm的cs文件
  5. 利用SEO，容易被搜索引擎收录！

2、选择器

作用：选择页面上的某一个或者某一类元素

2.1基本选择器

  1. 标签选择器，标签选择器，会选择到页面上所有的这个标签的元素
  2. 类选择器，格式: .class的名称{},可以多个标签同一个 class名称，用于归类
  3. Id选择器,格式：#Id名称，Id必须保证全局唯一！

优先级

不遵循就近原则，固定的

id选择器 >class选择器>标签选择器
